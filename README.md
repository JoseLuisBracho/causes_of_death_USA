# causes_of_death_USA

## Goal:
Identify the main causes of death in the United States

## Resources used:
* Data from data.gov
* Python / Jupiter
	* NumPy
	* Pandas
	* Matplotlib
	* GMaps
	* scipy.stats
* MS Excel

* There is not differences bewteen Northern and Southern states.
